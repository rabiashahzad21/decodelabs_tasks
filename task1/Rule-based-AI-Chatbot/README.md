# 🤖 SmartAssist AI - Rule-Based Chatbot

SmartAssist AI is a professional rule-based chatbot developed using HTML, CSS, and JavaScript. The chatbot responds to predefined user inputs using simple `if-else` logic, making it an excellent beginner AI project while demonstrating decision-making concepts.

## ✨ Features

- Modern ChatGPT-inspired user interface
- Rule-based conversation using if-else statements
- Handles greetings (Hi, Hello, Hey)
- Responds to AI and Machine Learning questions
- Help command
- Exit command (Bye/Exit)
- Typing animation
- New Chat button
- Responsive design
- Continuous conversation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Clone this repository.
2. Open the project folder in VS Code.
3. Open `index.html` using Live Server or your web browser.
4. Start chatting with SmartAssist AI.
